# Cadastro de funcionários.

> Desenvolvi esse projeto no intuito de aprender e mexer com DML e DQL (SQL) usando JDBC no Java. Utilizei o Docker para subir containers para rodar as ferramentas.
> Esse projeto setta funcionarios no banco de dados com encapsulamento, levantando esse set no banco de dados usando "INSERT VALUES".
> O projeto também consta com criações de Querys usando SELECT, retornando dados dos funcionários.
